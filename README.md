# DRDS
Python ve Streamlit ortamında geliştirilen prototip üzerinden yapılan karşılaştırmalı analizlerde, önerilen DRBS modelinin geleneksel statik yöntemlere kıyasla hedef noktaya varış süresini optimize ettiği ve müdahale ekiplerinin maruz kaldığı "Yol Risk Skoru"nu anlamlı ölçüde düşürdüğü hipotezi test edilmektedir.
